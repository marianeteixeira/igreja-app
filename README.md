# IgrejaApp

Sistema web para gerenciamento de membros e controle de presença em igrejas.

## Funcionalidades (MVP)
- Cadastro de membros
- Registro de presença diária
- Consulta de presença

## Futuras melhorias
- Relatórios (aniversários, frequência)
- Controle por setores
- Multi-igrejas
- Autenticação de usuários

## Tecnologias
- ASP.NET Core Web API
- Entity Framework Core
- SQL Server
